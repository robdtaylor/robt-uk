# robt.uk

Personal website built with Hugo and PaperMod theme.

## Local Development

```bash
hugo server -D
```

Then open http://localhost:1313

## Deployment

Deployed automatically via Cloudflare Pages on push to main.

## Writing Posts

Create new posts in `content/posts/`:

```bash
hugo new posts/my-new-post.md
```

Edit the file, set `draft: false` when ready to publish.
